CyberOS

AI-native internal operations platform — turning will into real.

Built by CyberSkill — a 10-person Vietnam-based software consultancy. CyberOS is the AI-native operating system for the company: 22 federated modules across five phased rollouts (P0 → P4), unified by a three-layer BRAIN, a persona-aware CUO orchestrator covering 47 C-roles with 221 workflows chaining 104 author+audit skill pairs, six cross-cutting infrastructure pillars (AUTH · AI Gateway · MCP Gateway · OBS · GraphQL Federation · NATS), and a Vietnamese-market wedge of localised skills. Three modules shipped today (memory · skill · cuo, with the CUO supervisor running Phases 1–4 live and 49/50 tests green); the rest land between P0 · exit and P4 · GA.

What is CyberOS

The substrate · the catalog · the orchestrator

CyberOS is built on three shipped modules that compose to form an AI-native operating system. Every other module in the catalog plugs into this triangle.

P0 · Shipped

BRAIN — the substrate

Append-only audit ledger

Local-first, audit-chained memory. Three layers: FS at the edge (.cyberos-memory/), pgvector + Apache AGE in the middle, archival corpus at the cold tier. MMR inclusion proofs, Ed25519-signed tree heads, deterministic export, GDPR purge.

Skill — the catalog

Portable agentic skills

Open Agent Skills standard (Anthropic · VS Code · Cursor · Codex). Rust host + Bun toolchain. 5 Vietnamese-market skills shipped (MST · VAT · VietQR · CCCD · legal); tax-filing is planned. Three-level progressive disclosure; 12/12 Python ↔ Rust parity tests green.

CUO — the orchestrator

Chief Universal Officer

The router. Natural language in, skill chain out, memory record on file. Genie is the mascot; CUO is the inner decision frame. 47 C-suite persona workflows load on demand. Notify / Question / Review ambient modes — never auto-acts on irreversible operations.

Compliance scope

Three concentric compliance rings

CyberOS plans for three jurisdictional concentric rings simultaneously. The inner ring (Vietnam) is the cornerstone — every CyberOS deployment respects it. The middle ring (cross-border) adds GDPR + Singapore PDPA when Singapore HoldCo flips at P3. The outer ring (international standards) lights up gate-by-gate.
