Functional Requirements Catalog

Every functional requirement across the 22 CyberOS modules, traced to the PRD §9 (Priority Module Deep-Dives) and the SRS §4 (Formal FR catalog). Each FR has an id, a normative description (MUST/SHOULD/COULD/MAY per BCP-14), a verification method, an owning module, and a phase.
