# product-metrics-review@1 — Quarterly product metrics review

A `product-metrics-review@1` artefact authors a quarterly product OKR + metrics review — DAU/WAU/MAU, retention cohorts, feature-adoption funnels, north-star metric movement, OKR attainment. Per Amplitude / Mixpanel / Pendo product-analytics best practices + Reforge product-strategy framework + Sequoia product-led-growth playbook.

## Required sections (template.md H2 order)
1. Reporting period & scope (in-scope surfaces/features/cohorts)
2. North-star metric movement (vs target + vs prior period)
3. DAU/WAU/MAU + DAU/MAU stickiness ratio
4. Retention cohorts (Day-1 / Day-7 / Day-30 by cohort)
5. Feature-adoption funnels (per-feature activation rate + drop-off)
6. OKR attainment (per-O quantitative scoring)
7. Top wins + top losses (with narrative)
8. Next-quarter focus
