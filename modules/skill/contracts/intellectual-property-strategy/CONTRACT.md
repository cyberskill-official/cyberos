# intellectual-property-strategy@1 — IP portfolio strategy

An `intellectual-property-strategy@1` artefact authors the annual (or per-event) IP portfolio strategy — patent filings, trademark portfolio, trade-secret protection plan, defensive vs offensive posture, defensive-publication strategy, FTO (freedom-to-operate) opinions.

## Format
- Per WIPO IP strategy framework + USPTO Patent Bar guidance + Madrid Protocol (trademarks) + UTSA / DTSA (trade secrets).
- Each line item links to an enforcement / defense rationale.

## Required sections (template.md H2 order)
1. Portfolio inventory (patents / trademarks / copyrights / trade secrets)
2. Strategic posture (defensive / offensive / both / blocking)
3. Patent filing plan (utility / design / provisional / international PCT)
4. Trademark filing plan (USPTO / Madrid / regional)
5. Trade-secret protection plan (UTSA / DTSA controls)
6. FTO (freedom-to-operate) assessment
7. Enforcement budget envelope
8. Risks & open issues

## Citations
- WIPO IP strategy framework
- USPTO Patent Bar Manual of Patent Examining Procedure (MPEP)
- Madrid Protocol for trademarks
- Uniform Trade Secrets Act (UTSA) + Defend Trade Secrets Act (DTSA)
- Patent Cooperation Treaty (PCT) for international filings

## KPI
- Filings completed vs plan
- Enforcement actions concluded
- IP-derived revenue (licensing, defense settlements)
