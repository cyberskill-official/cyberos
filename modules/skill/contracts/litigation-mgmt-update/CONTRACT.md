# litigation-mgmt-update@1 — Litigation management status update

A `litigation-mgmt-update@1` artefact authors a quarterly (or per-board-meeting) litigation status update. Covers active cases, exposure (potential damages), settlement posture, outside-counsel costs, board chapter content. Per the Litify case-management model + ACC Value Challenge benchmarking.

## Format
- One row per active matter; one summary table per matter type (litigation / arbitration / regulatory action / IP dispute / commercial dispute).
- Exposure quantified in USD; reserve recommendation included.

## Required sections (template.md H2 order)
1. Active matters inventory
2. Exposure roll-up (potential damages by matter type)
3. Settlement posture (open / mediation / settled / judgment)
4. Outside-counsel costs (per matter + cumulative; vs ACC Value-Challenge benchmark)
5. Reserve recommendation (vs accounting reserve)
6. Board chapter content (1-page summary)
7. Quarter-over-quarter delta narrative

## Citations
- Litify case-management model
- ACC Value Challenge for outside-counsel billing
- ABA Litigation Section best practices

## KPI
- Litigation exposure trend
- Outside-counsel cost vs budget
- Settlement-to-judgment ratio
