> Per `../../docs/The C-Suite Reference.md` §5.4 + §4 (9-block schema).

## §1  Identity & scope
- **Full disambiguated title:** Chief Brand Officer.
- **Persona slug:** `chief-brand-officer`
- **Reports to:** CEO (typically).
- **Stage prevalence:** Seed: — · Series A: — · Scale-up: optional · Growth: optional · Enterprise: optional (per §3 matrix).
- **One-sentence scope:** Brand stewardship; rare standalone — usually within CMO.

## §2-§4  Inputs (information / stakeholder / resource)
See C-Suite Reference §5.4 for the full input lists. Expand in next session.

## §5  Outputs
**Strategic:** brand strategy; brand-architecture; brand-extension theses. **Operational:** brand-guideline enforcement; campaign-brand review. **Communication:** brand book; agency briefs. **Team:** Brand + Creative + Design Systems.

## §6  Cadence
Daily / weekly / monthly / quarterly / annual rhythms per role profile. See §5.4.

## §7  KPIs
| Brand awareness | survey per market | survey |
| Brand favourability | survey | survey |
| Brand-consistency audit | quarterly | brand audit |

## §8  Audit criteria
- **Quantitative:** moves the persona's 3-5 KPIs above; forecast within ±10 %.
- **Qualitative rubric:** alignment / coherence / customer-grounding / risk-realism / communicability.
- **Universal failure modes (per §6):** playbook transplant; activity over outcomes; silo-ing; forecast drift without narrative; hero dependence; AI-washing.
- **Role-specific failure modes:** brand-bible-without-application; over-rebranding fatigue; brand-team siloed from product/marketing.
- **Commercial baseline reminders (per §8):** don't add a C-title for a process problem; match title to stage; disambiguate acronyms; beware hype-cycle titles; audit outputs not titles.

## §9  Tools & stack
Frontify / Brandfolder (DAM); Figma + Adobe CC (design); Sprout (social brand monitoring).
