## §1  Identity & scope
- **Full disambiguated title:** Chief Remote Officer.
- **Persona slug:** `chief-remote-officer`
- **Reports to:** CEO (typically).
- **Stage prevalence:** Peaked 2020-2022; mostly absorbed by CHRO post-2022 (per §3 matrix).
- **One-sentence scope:** Peaked 2020-2022, now embedded in CHRO.

## §5  Outputs
**Strategic:** distributed-work strategy; tools + culture for remote/hybrid. **Operational:** virtual-event coordination; remote-onboarding programs. **Communication:** remote-culture talks. **Team:** small — often embedded in People Ops.

## §6  Cadence
Per role profile in §5.7.

## §7  KPIs
| Remote-employee eNPS | > 30 | survey |
| Async-communication adoption | per ritual | survey + tool analytics |
| Distributed-onboarding NPS | > 30 | new-hire survey |

## §8  Audit criteria
- **Quantitative:** moves KPIs above; forecast within ±10 %.
- **Qualitative rubric:** alignment / coherence / customer-grounding / risk-realism / communicability.
- **Universal failure modes (per §6):** playbook transplant; activity over outcomes; silo-ing; forecast drift without narrative; hero dependence; AI-washing.
- **Role-specific failure modes:** remote-vs-hybrid policy whiplash; tool-bloat; missing the absorption-into-CHRO transition.
- **Commercial baseline reminders (per §8):** don't add a C-title for a process problem; match title to stage; disambiguate acronyms; beware hype-cycle titles; audit outputs not titles.

## §9  Tools & stack
