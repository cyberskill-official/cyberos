CyberOS — what it is and the bet

CyberOS is the AI-native internal operations platform that runs CyberSkill, a 10-person fully-remote Vietnamese software consultancy founded in 2020. It is engineered from day one to be sold to other small-to-mid consultancies in Phase 4. It replaces a fragmented stack (Notion, Slack/Zalo, Asana, HubSpot, Gmail, Excel-payroll, paper contracts) with one cohesive, agent-operable system.

The central nervous system is BRAIN — a per-tenant universal memory layer with three storage tiers. The visible surface is Genie, the company-mascot AI assistant that takes on the additional role of CUO (Chief Universal Officer): a persistent executive layer performing the ten C-level functions a 10-person company cannot afford to staff individually.

The bet, in one sentence: a 10-person Vietnamese boutique can punch globally if every Member's daily work happens in one AI-rich system whose memory is rich enough to substitute for the C-suite the company does not have. Three sub-bets sit underneath: (1) agent parity is the moat — every UI action has an MCP equivalent under the same RBAC; (2) CUO is the brand — a named, persistent, persona-versioned mascot beats a generic AI panel; (3) BRAIN is the substrate — the three-layer architecture keeps memory transparent, private, and rich enough that any AI consumer answers with cited provenance.

The platform is multi-tenant from day one to avoid refactor cost when the external commercial path opens at P4. It is built on Apollo Federation v2 (subgraph per module), Module Federation for the frontend, MCP 2025-11-25 for agent operability, PostgreSQL 17 with pgvector + Apache AGE + PGroonga for storage and search, Vietnamese (vi-VN) as the default locale with English (en-US) parity, and AWS Bedrock as the primary LLM provider with Anthropic and OpenAI as Zero-Data-Retention fallbacks.
