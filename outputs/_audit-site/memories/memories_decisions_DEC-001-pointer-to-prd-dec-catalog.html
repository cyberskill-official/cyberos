Extending the catalogue

A new decision lives at DEC-067+ (next free integer). A change to an existing decision is recorded as a superseding decision (DEC-NEW supersedes DEC-OLD) — the old row remains visible with status `Superseded`. Same pattern REW uses for parameter versions.

How to use this memory

When a feature request requires citing or revising a decision, look it up in the PRD/SRS by ID. Add a memory file at `memories/decisions/DEC-{N}-{slug}.md` only when there is a project-internal addendum or local interpretation worth recording — do not duplicate the canonical SRS Part 13 entry here.
