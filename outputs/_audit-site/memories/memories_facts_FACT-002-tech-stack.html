FACT-002 — CyberOS technology stack at a glance

Frontend host shell. Module Federation (DEC-001).

Backend gateway. Apollo Federation v2.5+ supergraph (DEC-002), per-module subgraph.

Database. PostgreSQL 17 with pgvector (HNSW), Apache AGE (graph), PGroonga (multilingual full-text), pg_jsonschema (DEC-003).

Eventing. NATS JetStream (DEC-004) on canonical subjects `{module}.{entity}.{verb}`.

Object store. S3-compatible — Hetzner S3 primary, MinIO for dev (DEC-005).

Auth. OAuth 2.1 + PKCE + JWT (DEC-006); MFA TOTP minimum, WebAuthn for elevated roles (DEC-007).

Multi-tenancy. Schema-per-tenant + Postgres RLS (DEC-008); per-tenant region pinning for residency (DEC-009).

LLM providers. AWS Bedrock primary (DEC-012); Anthropic + OpenAI Zero-Data-Retention failover (DEC-013).

Embeddings. BAAI/bge-m3 (DEC-014/041) — multilingual, native Vietnamese.

Reranker. bge-reranker-v2-m3 (DEC-017/042).

Chunking. Anthropic Contextual Retrieval (DEC-018).

Agent runtime. LangGraph topology (DEC-027 in CUO context); Anthropic Skills directory format for persona/skill files; persona registry under `~/.cyberos/skills/cuo/<role>/SKILL.md`.

MCP. Model Context Protocol 2025-11-25 spec; OAuth 2.1 + PKCE + PRM discovery; per-module MCP servers federated through one gateway.

Audit. Merkle-chained audit log (DEC-019); BRAIN filesystem layer is the local-edge expression of this same pattern.

Observability. LGTM stack — Loki, Grafana, Tempo, Mimir/Prometheus (DEC-021).

Orchestration & GitOps. Kubernetes + ArgoCD (DEC-022/023).

KMS. HashiCorp Vault; per-tenant KMS hierarchy (DEC-024).

Edge. Cloudflare for CDN, TLS, DDoS, WAF (DEC-025); tenant-subdomain DNS (DEC-026).

Desktop / Mobile. Tauri desktop (DEC-030); React Native mobile (DEC-031, evaluation only through P3).

PDF. Puppeteer (DEC-032).

Locale. vi-VN default with en-US parity (DEC-034). Be Vietnam Pro typography for diacritic-aware rendering.

Hosting. Hetzner primary; Vultr Singapore region; OVH for the eu-shard at P3+.

How to use this memory

Use this when answering "what does CyberOS use for X" or when proposing a new dependency. New tech choices need a DEC entry — they don't slot in informally. Reference the relevant DEC ID in any addition to this card.
