# Glossary

The glossary collects every term, acronym, and proper noun used across the CyberOS docs, grouped into five categories:

- CyberOS-specific
- Tech stack
- Regulatory / compliance
- Business / domain
- Vietnamese / language

Entries are sourced from AGENTS.md and the design system glossary.

The term entries themselves are rendered client-side by the interactive glossary page on the generated site, which colour-codes the five categories and offers a full-text search box plus an A-Z jump for navigation. The entry data is embedded in that page's script and did not survive the HTML-to-markdown migration, so it is not reproduced on this page.
