# CyberOS — Full Build Task Checklist (P0 → P4)

> **How to use:** Each phase must satisfy ALL exit criteria before the next phase begins.  
> Compliance tasks are **non-skippable** — technical done ≠ phase done.  
> Work within a phase runs in parallel across teams (see `cyberos_module_independence.md`).
>
> Status: `[ ]` = pending · `[x]` = done · `[~]` = in progress · `[b]` = blocked
